## Installation

Run the following commands in this directory.

With Docker : 
```shell
docker build -t frontend .
```

Without Docker : 
```
yarn install
```

## Usage

### Execute app

Run the following commands in this directory.

With Docker :
```shell
docker run frontend
```

Without Docker :
```
yarn start
```

## Contributing

The following commands do not concern Docker.

### Run tests

```
yarn test
```

### Apply code style

```
yarn lint
```

### Build production app

```
yarn build
```
